# AWS ec2 VS AWS Elastic Beanstalk
## ec2
- 클라우드 리소스 **제공**
- 인스턴스를 제공해줌
- IaaS(Infrastructure-as-a-Service)
## EB
- 여러개의 인스턴스(ec2같은거)를 가지며 알아서 오토스케일링을 해준다. 
- 인스턴스를 **관리**해줌
- PaaS(Platforms-as-a-Service, PaaS)
