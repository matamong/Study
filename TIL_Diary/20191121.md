# 20191121 TIL Diary

- Git 
   - `Fork`

   - `clone`하고 `remote` 설정
   ```git
   $ git clone [주소]
   $ git remote [이름(ex: upstream)] [주소]

   $ git remote -v
   ``` 
   - `branch` 생성
   ```git
   $ git branch [branch 이름]
   ```
   - 수정하고 `push`  (**바로 `master` 브랜치에 `push` 하지 않도록!!**)
   ```git
   $ git add [수정 파일]
   $ git commit [커밋할 파일]
   $ git push origin [브랜치명]
   ```
   
   - 본인 repository에서 `Pull Request`
