# **.gitignore 문제점 정리**

## **적용이 안 될 때** <br>
- ### .gitignore의 파일명을 정확하게 해주자.  (.gitignore.txt -> .gitignore)
- ### 파일이 적용되기 전에 리모트에 남아있는 캐쉬파일을 지워주자.
```git
$ git rm -r --cached .
$ git add .
$ git commit -m "fixed untracked files"
```
